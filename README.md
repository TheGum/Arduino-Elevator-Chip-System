# Arduino-Elevator-Chip-System

Simple elevetor system with chip controll + SD Card Database.
 
 - Configuration:
  * Arduino Uno
  * SD Card module
  * RFID MFRC522
  
 - Included libraries:
  * jwhiddon/EDB - https://github.com/jwhiddon/EDB
  * miguelbalboa/rfid - https://github.com/miguelbalboa/rfid
