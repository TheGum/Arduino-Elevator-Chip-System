# Elevator Access Control Card Reader

Simple Elevator Access Control Card Reader.
 
Configuration:
  * Arduino Uno
  * SD Card module
  * RFID MFRC522
  
Included libraries:
  * jwhiddon/EDB - https://github.com/jwhiddon/EDB
  * miguelbalboa/rfid - https://github.com/miguelbalboa/rfid
